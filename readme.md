## Задание
- Написать функцию <b>sumOfString</b>, которая возвращает число: сумму всех цифр строки <b>str</b>.
- Функция принимает:
  - str - строка.

## Примечание
- В строке могут быть положительные и отрицательные числа от -9, до 9.
- Все числа разделены запятыми.

## Примеры
- ('1,2,3,4,5') => 15
- ('') => 0
- ('-2,5,7,-9,3') => 4
- ('5,2,7,-9,-5,-4') => -4
